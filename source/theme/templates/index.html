The Full Stack Python Guide to Deployments is a new ebook launching on July 28 with detailed step-by-step tutorials for configuring and deploying Python web applications. There's even a version that is packaged with a nicely formatted copy of the Full Stack Python website.

All of the content is entirely new and can be used by itself or in combination with the material available on Full Stack Python.

{% include "email-for-book.html" %}

What's in the book?

We'll take an example open source Python web application through a complete deployment on a virtual private server. In each chapter, we will first learn how to deploy the application manually, to fully understand each step, then automate the steps with Fabric and Ansible.

Still have questions? Check out the frequently asked questions (FAQ) page for more information.
